# Инструкция по настройке экспорта в Google Sheets

## Что добавлено

Добавлена функциональность автоматической и ручной выгрузки отчетов в Google Sheets со следующими возможностями:

1. **Автоматический экспорт по расписанию** (раз в неделю или по вашему расписанию)
2. **Ручной экспорт через админ меню** (кнопка "📤 Экспорт в Google Sheets")
3. **Автоматическое создание новых таблиц** для каждого месяца за 3 дня до конца текущего месяца
4. **Умная проверка дубликатов** - бот отслеживает какие записи уже экспортированы и не дублирует их

## Формат таблицы

Каждая месячная таблица содержит следующие колонки:
- **Дата** - дата выполнения работы
- **Фамилия Имя** - ФИО работника
- **Место работы** - локация (поле/склад)
- **Вид работы** - тип выполненной работы
- **Количество часов** - отработанные часы

## Настройка переменных окружения (.env)

Добавьте в ваш файл `.env` следующие переменные:

```env
# OAuth клиент для Google API (путь к файлу oauth_client.json)
OAUTH_CLIENT_JSON=oauth_client.json

# Путь к файлу с токеном (создастся автоматически после первой авторизации)
TOKEN_JSON_PATH=token.json

# ID папки на Google Drive, куда будут сохраняться таблицы
# Чтобы получить ID папки:
# 1. Откройте папку на Google Drive
# 2. Скопируйте ID из URL: https://drive.google.com/drive/folders/[ВОТ_ЭТО_ID]
DRIVE_FOLDER_ID=ваш_folder_id

# Префикс для названия таблиц (по умолчанию "WorkLog")
EXPORT_PREFIX=WorkLog

# Включить автоматический экспорт (true/false)
AUTO_EXPORT_ENABLED=true

# Расписание автоматического экспорта в формате cron (5 чисел через пробел)
# Формат: минута час день месяц день_недели
# Примеры:
# 0 9 * * 1    - каждый понедельник в 9:00
# 0 0 * * 0    - каждое воскресенье в 00:00
# 0 18 * * 5   - каждую пятницу в 18:00
# 0 9 1 * *    - первого числа каждого месяца в 9:00
AUTO_EXPORT_CRON=0 9 * * 1
```

## Первый запуск

При первом запуске бота после добавления функционала экспорта:

1. Бот автоматически создаст новые таблицы в базе данных
2. При первом экспорте откроется браузер для авторизации Google OAuth (если еще не авторизованы)
3. Разрешите боту доступ к Google Drive и Google Sheets
4. Токен доступа сохранится в файл `token.json` и будет использоваться автоматически

## Использование

### Ручной экспорт (через админ меню)

1. Откройте бота и нажмите "🧰 Меню"
2. Выберите "⚙️ Админ" (доступно только администраторам)
3. Нажмите "📤 Экспорт в Google Sheets"
4. Бот экспортирует все новые записи, которые еще не были отправлены в таблицу
5. Получите уведомление о количестве экспортированных записей

### Автоматический экспорт

Если `AUTO_EXPORT_ENABLED=true`, бот автоматически будет экспортировать новые записи согласно расписанию `AUTO_EXPORT_CRON`.

По умолчанию: каждый понедельник в 9:00

### Создание таблиц для новых месяцев

Бот автоматически создаст таблицу для следующего месяца за 3 дня до конца текущего месяца. Например:
- 28-31 января → создается таблица для февраля
- 25-28 февраля → создается таблица для марта
- и т.д.

## Структура базы данных

Добавлены две новые таблицы:

### `google_exports`
Отслеживает какие отчеты уже были экспортированы:
- `report_id` - ID отчета
- `spreadsheet_id` - ID таблицы Google Sheets
- `sheet_name` - название листа (год-месяц)
- `row_number` - номер строки в таблице
- `exported_at` - дата экспорта
- `last_updated` - дата последнего обновления

### `monthly_sheets`
Хранит информацию о созданных месячных таблицах:
- `year` - год
- `month` - месяц
- `spreadsheet_id` - ID таблицы Google Sheets
- `sheet_url` - ссылка на таблицу
- `created_at` - дата создания

## Логирование

Все операции экспорта логируются:
- Успешные экспорты: `INFO: Exported X reports to YYYY-MM`
- Создание новых таблиц: `INFO: Created new sheet for YYYY-MM: [URL]`
- Ошибки: `ERROR: ...`

Проверяйте логи для отслеживания работы экспорта.

## Troubleshooting

### "Ошибка авторизации Google"
- Убедитесь, что файл `oauth_client.json` существует и содержит правильные данные
- Удалите `token.json` и попробуйте снова - откроется окно авторизации

### "Google API HttpError 403"
- Проверьте, что папка на Google Drive расшарена для вашего аккаунта
- Убедитесь, что у вас есть права на создание файлов в этой папке

### "Google API HttpError 404"
- Проверьте правильность `DRIVE_FOLDER_ID` в `.env`
- Убедитесь, что папка существует и доступна

### "Invalid cron expression"
- Проверьте формат `AUTO_EXPORT_CRON` - должно быть ровно 5 чисел/символов через пробел
- Используйте онлайн-генераторы cron выражений для помощи

## Пример настройки

Минимальный пример `.env`:

```env
TELEGRAM_TOKEN=ваш_токен_бота
ADMIN_IDS=123456789

# Google Sheets
OAUTH_CLIENT_JSON=oauth_client.json
TOKEN_JSON_PATH=token.json
DRIVE_FOLDER_ID=1a2b3c4d5e6f7g8h9i0j
EXPORT_PREFIX=WorkLog
AUTO_EXPORT_ENABLED=true
AUTO_EXPORT_CRON=0 9 * * 1
```

После этого настройте переменные и файлы в **backend** Terra App (см. **[README_APP.md](README_APP.md)**).

## Дополнительно

- Таблицы создаются в формате: `WorkLog_YYYY_MM` (например, `WorkLog_2025_10`)
- Заголовки таблиц автоматически форматируются жирным шрифтом
- Данные добавляются в таблицу по мере экспорта, старые записи не изменяются
- Каждый отчет экспортируется только один раз (отслеживается в БД)
