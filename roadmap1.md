Фаза 1: Таблица
этап 1. как должна выглядеть таблица
1. User ID- номер телефона который вводиться в начале при регистрации в боте (это уже и так есть)
2. Имя- имя введенное при регистрации (тоже уже есть)
3. Дата работы-фактическая дата работы (не путать с датой заполнения)
4. Часы-колл-во часов
6. Тип работ-какой тип работы выполнялся 
7. Тип Техники 
8.Техника (если трактор то модель трактора если камаз то просто камаз)
9. Вид деятельности для техники: (Заполняется только для тракторов, если в пункте 4.1 выбран КамАЗ, то в этом столбец автоматически, без заполнения данных работником, указывается - Перевозка)
10. Вид работы (только для ручной работы) 
11. Локция
12.Культура



фаза 2: последовательность и заполнение отчета в боте
1.основное меню- 1. ОТД, 2.Статистика, 3.Настройки (убрать кнопку приложение, выход в приложение только через кнопку возле клавиатуры)
2.дата работы (все гуд все так же и оставляем)
3.часы (тоже все гуд ничего не меняем)
4.тип работы (тоже все гуд)
4.1 тип техники (тоже все гуд)
4.2 выберите технику (тоже все гуд)
4.3(?) вид деятельности для техники- 1. Выравнивание, 2. К-ц(междурядная), 3. Боронование, 4. Сев, 5. Опрыскивание, 6.Пахота, 7. Чизелевание, 8.Уборка, 9.прочее, 10.Назад(сам ручками через приложение поменяю)
5(?).выберите работу (все гуд, поменяю ручками)
6. Выберите локацию - переименовать в поле (ПРИМЕЧАНИЕ: выбор локации должен быть у 
всех выглядеть одинаково, не важно работы у 
них на тракторе, КамАЗе или ручная)
7. Культура (все гуд) 
8. Проверка данных: кнопку назад заменить на "Отменить(в 
главное меню)"
